# Todo App - NestJS & React Native

This project is a full-stack Todo application built using NestJS for the backend and React Native for the frontend.  
It allows users to register, log in, and manage their daily tasks easily.  

---

## Project Description

The backend is developed with NestJS and uses Prisma ORM to interact with a PostgreSQL database.  
It provides RESTful APIs for user authentication and task management.  
The frontend is a simple React Native app that connects to the backend, allowing the user to log in, view, and add tasks dynamically.  

---

## Features

### Backend
- Built with NestJS and TypeScript  
- User authentication using JWT  
- Secure password hashing with bcrypt  
- CRUD operations for tasks  
- Prisma ORM with PostgreSQL  
- CORS enabled for frontend integration  

### Frontend
- Built with React Native and TypeScript  
- User login screen with email and password  
- Todo screen to add and list user-specific tasks  
- Token-based authentication  
- Clean and simple UI
